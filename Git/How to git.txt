Pull: Hämta ändringar som finns.
Push: Ladda upp ändringar i repository.

Git bash commands:

…or create a new repository on the command line:

echo "# howto" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/musslan91/howto.git
git push -u origin master

…or push an existing repository from the command line:

git remote add origin https://github.com/musslan91/howto.git
git push -u origin master

Kör aldrig två masters i ett repo utan du vet vad du gör!!!
